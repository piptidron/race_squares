The program of the simple game "Squares_Race".

Six squares of different colours take part in the race.

1) First, the program will prompt you to choose the color of the square from these colours: "red","orange","yellow","green","blue", "purple".
2) Then the race to start.

If your square comes first to the finish, program says "You have Won!"
If not, then program says "You have Lost!"


Only one square can reach the finish line first.
 
I hope it will be your square,
Good luck!
